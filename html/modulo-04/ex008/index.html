Atributos De Formulários

O que aprendi de novo?

atributos da tag <form>

autocomplete que pode ser off ou on

action ação que acontece depois de clicar no submit é onde é colocado o nome do arquivo onde serão armazenados as informações recolhidas


ainda é um formulário simples

Nome

Cidade

Nota;

Quando o auto complete esta no off nao aparecem sugestões na caixa de formulário, por padrão o autocomplete fica on.

O action recebe nome de arquivo de uma linguagem de programação como por exemplo java javascript php etc.
