métodos GET e POST em formularás

GET e POST

o método get mostra as informações de cadastro na url.

o método post nao mostra as informações de cadastro na url mais podem ser acessadas facilmente pelo dev tool por exemplo..

Nota:

para contas bancarias, senhas, números telefónicos usa-se o HTTPS.

Usamos;

get quando os dados não sao cessível isto nao tem nada comprometedor no formulário

post usamos quando o numero de dados recolhidos ultrapassar 3000bts ou quando quanto o formulário tiver dados cessível.
