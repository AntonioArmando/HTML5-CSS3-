configurando iframe

dentro do iframe coloca-se a tag a para caso o navegador nao nao suporte o iframe exiba o link do conteúdo

exemplo de link no iframe

    <iframe src="" frameborder="0">
        <a href="link">visite o meu site
    </iframe>

o conteúdo do link so ira aparecer se o navegador nao for compatível com iframe.

configurações de borda iframe

As bordas do iframe no html so recebem dois parâmetros que são 0 e qualquer numero sendo que o;
0 (zero) significa sem borda
qualquer numero significa borda de 1px.
O tamanho das bordas do iframe podem se alteradas usando o style.
