Mídia quarry

sao usados para especificar o tipo de mídia que sera exibido o conteúdo.

existem vários tipos de mídia quarry e os que serão tratados neste documento sao as mídia "screen" e as mídia "print"

Mídia Screen

É nelas que sao feitas as configuradões do css para telas te celular, pcs, tv, relógios, etc

Mídia Print

É nelas que sao feitos as configuradões para a impressão

Notas;

este é o primeiro documento em html escrito por mim com suporte a dois tipos de mídia.
