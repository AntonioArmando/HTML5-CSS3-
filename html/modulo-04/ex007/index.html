Formulário Em HTML

para criar um formulário em html usa-se a tag <form>.
que recebe o elemento input que tem valiosos elementos como por exemplo;

  • <input type="image" src="" alt="">
  • <input type="email" name="" id="">
  • <input type="text" name="" id="">
  • <input type="submit" value="">
