Elementos Color e Range

color cria uma caixa para selecionar cores em um formulário.

range cria um livel uma barra que pode ser ajustado de acordo com os valores indicados. O valor máximo do nível é 100.
E também podemos colocar o valor inicial usando o atributo valor.

file permite enviar arquivos como imagens, audio, video etc. Mais isso torna obrigatório usar o método post.

Formulário

range cria um nível.

file permite enviar os arquivos no formulário.

color cria um seletor de cores no formulário.
